## Shenzhen Banying Suixing Technology Co., Ltd.
**Large Language Model Algorithm Intern**  
*Shenzhen, China · Jun. 2025 - Sep. 2025*

I worked on the model and systems path for an AI voice-interaction terminal designed for emotional companionship and learning support. The product combined real-time visual perception, speech synthesis, backend services, and camera hardware, so the work required balancing model quality, latency, and deployment reliability.

**What I did**

- Built a low-latency finger-reading pipeline using Google MediaPipe hand landmarks and Baidu PaddleOCR.
- Used fingertip location to prioritize text regions, reducing unnecessary OCR work and improving responsiveness.
- Debugged ESP32-CAM video streaming and connected camera input with server-side recognition.
- Containerized perception services with Docker for repeatable deployment.
- Fine-tuned fish-speech TTS behavior to improve intonation and emotional expressiveness.
- Improved speech-service concurrency performance by **70%** through streaming processing.

**Technical themes:** LLM applications, OCR, hand tracking, TTS, Docker, ESP32-CAM, edge-device integration.

## 深圳伴影随行科技有限公司
**大语言模型算法实习生**  
*中国深圳 · 2025.06 - 2025.09*

我参与了面向情感陪伴和学习辅助场景的 AI 语音交互终端研发。这个产品把实时视觉感知、语音合成、后端服务和摄像头硬件连接在一起，因此工作重点不是单个模型指标，而是模型质量、响应延迟和部署可靠性的平衡。

**主要工作**

- 使用 Google MediaPipe 手部关键点和百度 PaddleOCR 构建低延迟指读流程。
- 利用指尖位置优先筛选附近文本区域，减少无关 OCR 计算并提升响应速度。
- 调试 ESP32-CAM 视频流，把设备端摄像头输入接入服务端识别链路。
- 使用 Docker 容器化感知服务，提升部署一致性和复现性。
- 调整 fish-speech TTS 的语调与情绪表达，让语音反馈更自然。
- 通过流式处理将语音服务并发性能提升 **70%**。

**技术主题：** LLM 应用、OCR、手部追踪、TTS、Docker、ESP32-CAM、边缘设备集成。
